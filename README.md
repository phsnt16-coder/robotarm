
# SLU(SMART- Load & Unload): 2026 융합종합설계


본 프로젝트는 실시간 처리 기술과 수학적 적재 최적화 알고리즘을 결합한 지능형 로봇암 시스템이다. ArUco 마커 기반의 3차원 객체 포즈 추정(Pose Estimation) 데이터를 바탕으로 공간 효율을 극대화하는 3D Bin Packing 알고리즘을 실시간으로 수행한다. 이를 위해 Raspberry Pi 5의 멀티코어를 활용한 멀티프로세싱 큐 아키텍처를 구현한다. 이를 통해 상위 연산(Raspberry Pi)과 하위 구동(모터 제어) 간의 데이터 병목을 제거하고 안정적인 제어를 보장한다.



# Motivation 

지금까지의 로봇암 시스템은 주로 사전에 정의된 작업 시퀀스를 반복 수행하는 방식으로 운영된다. 그러나, 물품과 적재 공간의 크기가 다양한 환경에서는 기존 방식이 한계를 드러낼 수 있다. 고정된 적재 패턴에 의존하는 구조에서는 적재 물품의 크기 역시 고정되어 있기 때문에 효율이 저하되거나, 배치에 실패할 위험이 있다.  
이러한 한계를 극복하고자, 본 프로젝트에서는 적재된 물품의 위치 정보를 기반으로 남은 공간을 계산하고, 새로운 물품이 진입할 때마다 배치 가능 후보 위치를 재탐색하여 최적에 근접한 위치를 결정한다.



# Features

- UART 기반의 통신 프로토콜  
- 휴리스틱 알고리즘 기반의 최적의 적재좌표 계산  
- ArUco 마커 자동인식을 통한 파지좌표 계산
